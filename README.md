## Introduction
Security Operations Center(SOC) is a central hub within an organisation who is responsible for:
* Monitoring 
* Detecting 
* Responding 

to cyber threats for the organisation (IBM, 2021). This report will show my investigation into into the Boss of the SOCv3(BOTSv3) dataset. This dataset is a Splunk based capture the flag(CTF). Its aim is to simulate a realistic cyber incident which contains collection of logs including 

* network
* endpoint
* email
* cloud data from services like Amazon AWS and Microsoft Azure
